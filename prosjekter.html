Skoleprosjekt – Rekonstruksjon av OsloMet's nettsider

Første semester i studiet rekonstruerte jeg og min eksamensgruppe, OsloMet sin nettside for emnet matematisk modellering. Dette ga oss nyttig kunnskap og erfaring innenfor HTML, CSS og JavaScript. Vi vektet også universell utforming stort, og brukte sematisk HTML og CSS for best mulig utforming.
